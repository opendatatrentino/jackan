**About**

Java client library for CKAN catalogs. Features:

  * dependency handling with Maven
  * liberal Apache 2.0 license
  * well documented
  * unit tested with proper integration tests
  * uses ckan api v3 
  * supports Java 6+

**Usage**: See [docs](docs)

**Roadmap**: See [project milestones](../../milestones)
